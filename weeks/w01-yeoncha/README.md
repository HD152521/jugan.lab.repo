# w01 - 연차술사

> 이 폴더는 **이번 주 목표와 계획**을 적는 곳. 실제 앱 코드는 [`apps/yeoncha`](../../apps/yeoncha)에 있다.

## 이번 주 목표
- weekly.ship 시즌1의 첫 실험을 "완성해서 배포"까지 끝낸다.
- 직장인이 1초 만에 "내 연차로 며칠 쉴 수 있나"를 확인하고 공유하게 만든다.

## 뭘 만들까 (계획)
- **핵심 질문 1개에 집중**: "연차 N개 → 최장 연휴 며칠?"
- 입력은 딱 두 개 — 연차 개수(1~5), 기간(전체/2026/2027).
- 출력은 연휴 길이순 카드. 각 카드에 연차 쓰는 날짜, 효율(연차 1개당 일수), D-day.
- 바이럴 훅: **공유하기**와 **캘린더(ICS) 등록**으로 마찰 없이 퍼지게.
- 데이터는 2026 하반기~2027 공휴일. 임시공휴일은 발표 시 추가.

## 만드는 순서
1. 공휴일 데이터 확정 (`holidays.js`) — 대체공휴일 포함, 근로자의 날 제외
2. 연휴 탐색 알고리즘 (`optimizer.js`) — 극대 연휴 구간 탐색, 순수 함수로 테스트 가능하게
3. UI (`index.html` + `app.js`) — 카드/공유/ICS
4. 순수 함수 테스트 (`node --test`)
5. Vercel 배포 후 `apps/hub` 링크 연결
