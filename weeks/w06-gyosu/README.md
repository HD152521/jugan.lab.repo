# w06 - 교수님 이메일 변환기

> 이 폴더는 **이번 주 목표·계획**. 실제 앱 코드는 `apps/gyosu`(예정)에 둔다.
> 타깃: 대학생 · hub 실험 #6 · **시즌: 중간/기말 시험기간(10월·12월·4월·6월)에 맞춰 출시**

## 한 줄 컨셉
"과제 늦었는데 봐주세요"를 붙여넣으면 격식 갖춘 교수님용 메일로 변환. before/after 한 방.

## 관심 끄는 축 (주력 = 바이럴 + 시즌 리텐션)
- 🚀 **바이럴(주력): before/after 대비** — 반말 초안 → 완벽 존댓말 메일 전환 장면이 릴스 훅 그 자체.
- 🔁 리텐션(보조): **내 템플릿 보관함** — 출석 문의/성적 문의/기한 연장 등 상황별 저장 → 시험기간마다 재방문이 구조적으로 발생.

## 유입(모집) 전략
- 릴스 훅: "교수님께 이렇게 보내면 큰일남" → 실제 변환 장면 (오리가 식은땀 → 안도)
- 시즌 타이밍: 시험기간 2주 전 출시 + 시험기간마다 리포스트
- 공유 장치: "친구한테 보내주기" — 메일 못 쓰는 친구 태그 유도

## 핵심 기능
- 상황 선택(과제 연장/성적 문의/출석/추천서 부탁/질문) + 핵심 내용 입력 → 완성 메일 생성
- 규칙 기반 템플릿 + 치환 (호칭/소속/과목명) — LLM 없이 시작, 필요 시 확장
- 제목 자동 생성 ("[과목명] 홍길동 — 과제 제출 기한 문의")
- 복사 버튼 + 내 템플릿 저장 (localStorage)

## 저장 방식
- 로컬로 충분 (템플릿 보관함 = localStorage)

## 완료 기준 (DoD)
- [ ] 상황 5종 템플릿, 대표 시나리오가 자연스러운 메일을 냄
- [ ] 템플릿 저장/불러오기
- [ ] 모바일 320px 깨짐 없음

## 리스크
- 너무 정형화된 문장 → 교수 입장에서 "복붙 티" → 변형 옵션 2~3안 제공
- 대필 논란 여지 → "예의 형식을 돕는 도구" 포지셔닝, 내용은 본인이 쓰게
