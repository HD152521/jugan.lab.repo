# mascot — 안경 오리 포즈 라이브러리

주간실험실 마스코트(안경 오리, 크루드 손드로잉)의 재사용 포즈 라이브러리.
출처: JuganLab 노션 캐릭터 시트. 매주 새로 그리지 않고 **여기서 골라 쓴다**.

## 파일

- `오리모음집(1~3).jpeg` — 원본 시트 3장. 각 1024×1024에 3×3 격자, 총 27포즈
- `duck/imgN_<표정>.png` — 낱장 포즈 27개. **328×328, 배경 투명, 셀 테두리 제거**

> **2026-07 갱신:** 낱장이 흰 배경 → **투명 배경**으로 교체됨. 파일명·크기(328×328)는
> 그대로라 기존 참조는 수정 없이 그대로 동작한다.

## 사용 규칙 (노션 킷)

- 안경 = 정체성 앵커, 어떤 포즈에서도 유지됨.
- 색 3색 이내(노랑·검정·주황). 팔레트: 딥그린 배경 + 옐로우 포인트.
- **배경 투명이라 딥그린 위에 바로 얹으면 된다.** 크림 카드는 이제 디자인 선택이지
  제약이 아니다. (구버전 낱장은 흰 배경이라 카드 위 배치가 필수였다.)
- 영상엔 오리가 반드시 등장(브랜드 핵심). `shared/video-pipeline`이 이 자산을 합성.

## 낱장 추출 방법 (재현용)

원본 시트에서 낱장을 다시 뽑아야 할 때의 기준. 눈대중 크롭보다 이 방식이 안전하다.

1. **격자 좌표는 측정해서 쓴다.** 1/3 등분이 아니다. 실측 경계선:
   - 행 `(26,337) (356,668) (687,1001)`
   - 열 `(26,338) (355,668) (685,997)`
   - 여기서 안쪽으로 6px 더 들어가야 둥근 테두리가 완전히 빠진다.
2. **배경 제거는 셀 가장자리에서 시작하는 플러드 필**로 한다.
   전역 흰색 임계값을 쓰면 **오리 눈 안쪽 흰자가 같이 뚫린다.**
   플러드 필은 바깥과 이어진 배경만 지우므로 눈이 살아남는다.
3. 알파 경계는 0.6px 가우시안으로 살짝 페더링한다. 축소 합성 시 계단현상 방지.
4. 알파 바운딩 박스로 트림한 뒤 328×328 캔버스 중앙에 배치한다(여백 8px).

스크립트: `yeoncha-hf` 프로젝트의 `scratchpad/extract_ducks.py`
(Pillow + numpy만 있으면 됨).

## 알려진 한계

- 원본이 JPEG라 라인아트 주변에 압축 잡음이 약간 있다. 확대 사용 시 눈에 띌 수 있다.
- `img2_감동`, `img3_자랑` 등 반짝임(✨) 장식이 포즈에 붙어 있다. 장식 없는 버전이
  필요하면 별도 리터치가 필요하다.
- 그림자가 그려진 포즈(`img1_현타`, `img3_신남b`)는 그림자도 함께 남는다.
  투명 배경 위에 떠 있는 형태라 어두운 배경에서는 어색할 수 있다.
