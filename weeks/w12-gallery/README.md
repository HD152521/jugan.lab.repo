## 관심 끄는 축 (주력 = 신뢰 자산 + 참여 리텐션)
- 🚀 **바이럴(주력): 투명한 결산** — "제일 잘된 건 의외로 X였습니다 / 3개는 망했습니다". 성공보다 실패 공개가 더 돈다.
- 🔁 **리텐션(주력): 다음 시즌 투표** — 방문자가 시즌2 아이디어에 투표. 결과가 다음 콘텐츠가 되므로 "내 투표 어떻게 됐나" 재방문.

## 유입(모집) 전략
- 릴스 훅: 12주 타임랩스 (그리드가 채워지는 애니메이션) → "총 사용자 N명, 최고는 OO"
- 결산 시리즈: 실험별 성적표를 캐러셀로 — 저장·공유 둘 다 노림
- 투표 참여가 곧 팔로우 이유: "시즌2 1화가 네 투표로 정해짐"

## 핵심 기능
