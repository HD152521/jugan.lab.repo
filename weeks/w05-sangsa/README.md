# w05 - 상사 말 번역기

> 이 폴더는 **이번 주 목표·계획**. 실제 앱 코드는 `apps/sangsa`(예정)에 둔다.
> 타깃: 신입·주니어 직장인 · hub 실험 #5

## 한 줄 컨셉
"간단하게 해줘" → "간단하지 않음". 상사의 애매한 말을 실제 의미로 번역하는 반유머 사전.

## 관심 끄는 축 (주력 = 바이럴)
- 🚀 **바이럴(주력): 상황극 릴스 최적 소재** — 오리가 상사 말을 듣고 멘붕하는 상황극. 번역 결과 카드("간단하게 해줘 = 오늘 안에 해줘") 공유가 곧 유입.
- 🔁 리텐션(보조): **사용자 제보 사전** — 방문자가 번역 제보 + 좋아요 투표. "이번 주 신규 번역 TOP 5"가 매주 갱신되는 구조.

## 유입(모집) 전략
- 릴스 훅: 소리 없이 자막만으로 웃기는 2컷 상황극 ("상사: 편할 때 와" → 오리: 지금 당장 달려감)
- 공유 장치: 번역 카드 이미지 저장 → 회사 단톡/스토리 공유 유도
- 댓글 유도: "너네 상사 최고 어록 댓글로" — 제보가 곧 콘텐츠 재료가 되는 선순환

## 핵심 기능
- 번역기: 입력(또는 목록에서 선택) → 실제 의미 + 위험도 게이지 (규칙 기반 사전, LLM 아님)
- 기본 사전 30~50개 내장 + 제보 폼 (승인제)
- 좋아요 순 정렬 "명예의 전당"
- 공유 카드 생성 (연차술사/축의금 이미지 카드 패턴 재사용)

## 저장 방식
- 순간 소비형 → 기본 로컬. 제보/좋아요만 가벼운 백엔드(서버리스 KV 또는 Supabase 테이블 1개)

## 완료 기준 (DoD)
- [ ] 기본 사전 30개 이상, 번역 카드 공유 동작
- [ ] 제보 → 관리자 승인 → 노출 흐름
- [ ] 모바일 320px 깨짐 없음

## 리스크
- 유머 톤 조절 실패 시 "직장 비하"로 읽힐 수 있음 → 자학 톤 유지, 특정 직군·성별 스테레오타입 금지
- 제보 스팸 → 승인제 + 길이 제한
