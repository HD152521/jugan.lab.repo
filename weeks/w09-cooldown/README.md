# w09 - 싸움 카톡 쿨다운

> 이 폴더는 **이번 주 목표·계획**. 실제 앱 코드는 `apps/cooldown`(예정)에 둔다.
> 타깃: 커플 · hub 실험 #9

## 한 줄 컨셉
화났을 때 쓴 메시지를 보내기 전에 순화 버전을 제안. "이 말 보내면 전쟁 납니다" 경고.

## 관심 끄는 축 (주력 = 바이럴/공감)
- 🚀 **바이럴(주력): 커플 공감 상황극** — 새벽 감성 + 싸움 직전의 긴장. 커플 계정들이 퍼가기 좋은 소재.
- 🔁 리텐션(보조): **월간 관계 리포트** — "이번 달 전쟁 3번 방지". 기록이 쌓이면 커플끼리 보여주는 재미.

## 유입(모집) 전략
- 릴스 훅: 분노의 장문 타이핑 → 전송 직전 → "전쟁 경보" 팝업 (오리가 뜯어말림) → 순화 버전
- 태그 유도: "이거 필요한 사람 태그" — 커플 태그가 자연 유입
- 시즌: 기념일 시즌(5월, 12월) 앞뒤로 리포스트

## 핵심 기능
- 메시지 입력 → 위험도 게이지(전쟁 확률 %) + 트리거 단어 하이라이트 ("항상", "너는 맨날", "됐어")
- 순화 버전 제안: 감정은 유지하되 공격 표현만 교체 (규칙 기반 사전)
- 10초 쿨다운 타이머: "그래도 보낼래?" 이중 확인
- 이번 달 참은 횟수 통계

## 저장 방식
- 로컬 전용 (메시지 내용 서버 전송 없음 — 민감 데이터, 신뢰 포인트로 명시)

## 완료 기준 (DoD)
- [ ] 트리거 사전 30개+, 대표 시나리오 5개에서 순화 제안이 자연스러움
- [ ] 위험도 게이지 + 쿨다운 흐름 동작
- [ ] 모바일 320px 깨짐 없음

## 리스크
- 순화가 어색하면 조롱거리 → 제안은 2안 제공 + "직접 고치기" 기본
- 관계 훈수 톤 금지 → 유머 유지, 심리 상담 아님 명시
