# w11 - 야근 수당 카운터 (예비 슬롯)

> 이 폴더는 **이번 주 목표·계획**. 실제 앱 코드는 `apps/yageun`(예정)에 둔다.
> 타깃: 직장인 · hub 실험 #11 · **예비 슬롯 — 반응 좋은 실험의 고도화나 신규 아이디어와 교체 가능**

## 한 줄 컨셉
퇴근 시간이 지나면 내 시급 기준으로 "회사가 나한테 빚지는 돈"이 초 단위로 올라간다.

## 관심 끄는 축 (주력 = 바이럴)
- 🚀 **바이럴(주력): 올라가는 숫자 화면** — 어두운 사무실 + 실시간 카운터. 릴스 첫 프레임 훅으로 최강.
- 🔁 리텐션(보조): 월간 누적 야근 시간/금액 리포트 — "이번 달 회사가 나한테 빚진 돈".

## 유입(모집) 전략
- 릴스 훅: "지금 이 회의, 회사가 나한테 47,200원 빚지는 중" — 숫자가 올라가는 화면 그대로
- 공감 타이밍: 수요일 밤 9~11시 업로드 (야근 중 스크롤하는 타깃)
- 공유 장치: "오늘의 야근 영수증" 카드 (시간·금액·치킨 환산)

## 핵심 기능
- 최초 설정: 월급 or 시급 + 정시 퇴근 시간 → 이후 원탭 시작
- 실시간 카운터 (초당 금액 상승 애니메이션) — 화면 자체가 콘텐츠
- 야근 종료 → 오늘의 영수증 카드 (공유용)
- 월간 누적 통계

## 저장 방식
- 로컬로 충분 (월급 정보는 민감 → 서버 전송 없음 명시)

## 완료 기준 (DoD)
- [ ] 시급 계산 정확 (월급→시급 환산 209시간 기준)
- [ ] 카운터 애니메이션 + 영수증 카드 공유
- [ ] 모바일 320px 깨짐 없음

## 리스크
- "그래서 수당 받을 수 있음?" → 법적 도구 아님 명시, 유머·현타 도구로 포지셔닝
- 예비 슬롯이므로 w01~w10 반응 데이터 보고 진행 여부 최종 결정
