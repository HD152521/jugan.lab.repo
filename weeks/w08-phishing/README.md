# w08 - 보이스피싱 문자 판별기

> 이 폴더는 **이번 주 목표·계획**. 실제 앱 코드는 `apps/phishing`(예정)에 둔다.
> 타깃: 부모님 세대 (공유하는 사람은 자녀 → 이중 도달) · hub 실험 #8

## 한 줄 컨셉
의심 문자를 붙여넣으면 사기 확률과 근거를 알려준다. "부모님 폰에 깔아드리세요".

## 관심 끄는 축 (주력 = 바이럴/공익 + 뉴스 사이클 리텐션)
- 🚀 **바이럴(주력): 효도 공유** — 실제 사용자는 부모님, 공유하는 사람은 자녀. "부모님께 보내드리세요" CTA가 도달을 2배로.
- 🔁 리텐션(보조): **"이번 주 신종 수법" 코너** 주간 갱신 — 피싱 뉴스가 뜰 때마다 트래픽이 다시 도는 구조.

## 유입(모집) 전략
- 릴스 훅: 실제 피싱 문자 예시 화면 → "이거 우리 엄마한테 왔던 거" 공감 → 판별 데모
- 시즌/이벤트: 명절(택배 사칭)·연말정산(환급 사칭) 시기마다 해당 수법 콘텐츠 재발행
- 공익 프레이밍: 저장·공유가 "좋은 일"이 되는 구조라 알고리즘 부스트 기대

## 핵심 기능
- 문자 붙여넣기 → 위험도(안전/주의/위험) + 근거 하이라이트 (단축 URL, 기관 사칭 문구, 금전 요구, 발신번호 패턴)
- 규칙 기반 패턴 매칭 (LLM 아님 — 오프라인 동작, 개인정보 서버 전송 없음이 신뢰 포인트)
- 유형별 실제 사례 아카이브 + "이번 주 신종 수법"
- 큰 글씨 모드 (부모님 세대 필수)

## 저장 방식
- 로컬 전용, 서버 전송 없음 명시 (신뢰가 곧 기능)

## 완료 기준 (DoD)
- [ ] 대표 피싱 유형 10종 탐지, 정상 문자 오탐 최소화 확인
- [ ] 판별 근거가 문장 내 하이라이트로 보임
- [ ] 큰 글씨 모드 + 모바일 320px

## 리스크
- 오탐/미탐의 책임 문제 → "참고용, 의심되면 118/112" 고지 필수
- 패턴 노후화 → 수법 DB를 데이터 파일로 분리해 주간 갱신 쉽게
