# w03 - 참은 지 N일 (전 연인 금연앱)

> 이 폴더는 **이번 주 목표·계획**. 실제 앱 코드는 `apps/chamgi`(예정)에 둔다.
> 타깃: 이별러 · hub 실험 #3

## 한 줄 컨셉
전 연인에게 연락 안 한 날을 금연앱처럼 카운트한다. "참은 지 D+37일 🔥"

## 관심 끄는 축 (주력 = 바이럴)
- 🚀 **바이럴(주력): 공감·밈** — "전 애인 연락 참은 지 37일" 카운터 + 공유 카드. 단톡방/스토리에 자조 개그로 퍼지기 좋음.
- 🔁 **리텐션(보조): 매일 체크 스트릭** — 금연앱 UX. 매일 열어 "오늘도 참았다", 연속 기록이 쌓이는 맛.

## 이번 주 목표
- "연락하고 싶은 충동"을 넘기게 도와주는, 웃기지만 은근 위로되는 도구.
- 카운터 + 공유 카드 + "참기 도와주기" 한 흐름 완성.

## 핵심 기능
### 카운터 (바이럴 훅)
- 시작일(마지막 연락일) 설정 → 실시간 **D+N일 / 시간** 카운트업
- 큰 숫자 + 상태 문구("초반이 고비", "이제 좀 살 만하죠?")
- **공유 카드**: "나 전 애인 연락 참은 지 N일째" PNG (연차술사 이미지 카드 재사용)
- 마일스톤 뱃지: 3일/7일/30일/100일/1년 → 달성 시 축하 + 공유 유도

### 참기 도와주기 (리텐션 + 실용)
- **"연락하고 싶어" 버튼** → 누르면: 참을 이유 랜덤 문구 / 흑역사 리마인더(내가 미리 적어둔 "헤어진 이유") / 대체 행동 추천
- **"못 참고 연락함" 버튼** → 스트릭 리셋 + "괜찮아 다시 시작" (죄책감 X, 금연앱 톤)
- 최고 기록 저장 → 리셋해도 "최고 N일" 남음

### 재미 요소
- 여러 명 관리 가능(전 애인 목록 ㅋㅋ) — 각각 카운터
- 익명·전부 로컬 저장 (민감 정보라 서버 X 강조)

## 만드는 순서
1. 카운터 로직(시작일 → D+N, 실시간 갱신) + 스트릭/최고기록 상태
2. UI: 큰 카운터 + 상태 문구 + 마일스톤
3. "연락하고 싶어/못 참음" 인터랙션 + 내 흑역사 메모
4. 공유 카드(canvas PNG)
5. 다중 대상 관리

## 완료 기준 (DoD, 가볍게)
- [ ] 시작일 설정 → 정확한 D+N 카운트
- [ ] 리셋/최고기록 동작
- [ ] 공유 카드 생성
- [ ] 톤이 위로/유머 (진지·훈계 금지), 전부 로컬 저장

## 데이터/기술 메모 & 리스크
- **민감성**: 이별·정신건강을 가볍고 따뜻하게. 절대 훈계·판단 X. 심각한 경우엔 유하게 넘어가기
- 시간 계산은 로컬 타임존, 자정 기준 일수
