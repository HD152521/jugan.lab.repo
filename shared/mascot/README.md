# mascot — 안경 오리 포즈 라이브러리

주간실험실 마스코트(안경 오리, 크루드 손드로잉)의 재사용 포즈 라이브러리.
출처: JuganLab 노션 캐릭터 시트. 매주 새로 그리지 않고 **여기서 골라 쓴다**.

## 파일
- `오리모음집(1~3).jpeg` — 원본 3×9 그리드 (총 27포즈)
- `duck/imgN_<표정>.png` — 그리드에서 낱장으로 크롭한 개별 포즈 (328×328, 흰 배경)

## 포즈 → 릴스 비트 매핑 (자주 쓰는 것)
| 비트 | 포즈 파일 |
|------|-----------|
| 훅·문제제기(멘붕) | `img1_멘붕` |
| 자학 공감(현타/지침) | `img1_현타`, `img3_한숨`, `img3_답답` |
| 빌드 몽타주(코딩중) | `img1_코딩` |
| 결과·성공(엄지척/뿌듯) | `img1_엄지척`, `img3_엄지척b`, `img3_자랑` |
| 중반 반전(충격) | `img1_충격` |
| 기능 설명(포인팅) | `img1_포인팅`, `img3_옆가리키기`, `img3_자신만만` |
| CTA·댓글 유도 | `img1_포인팅` (+ 아래 화살표 얹기) |
| 하이프·비트드롭(신남) | `img1_신남`, `img3_신남b`, `img2_축하` |
| 리액션 | `img2_하트`, `img2_울음`, `img2_화남`, `img1_궁금` |

## 사용 규칙 (노션 킷)
- 안경 = 정체성 앵커, 어떤 포즈에서도 유지됨.
- 색 3색 이내(노랑·검정·주황). 팔레트: 딥그린 배경 + 옐로우 포인트.
- 흰 배경이라 딥그린에 얹을 땐 **크림 카드 위에 배치**(키잉 불필요) 권장.
- 영상엔 오리가 반드시 등장(브랜드 핵심). `shared/video-pipeline`이 이 자산을 합성.
