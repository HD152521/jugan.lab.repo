# w07 - 야식 저금통

> 이 폴더는 **이번 주 목표·계획**. 실제 앱 코드는 `apps/yasik`(예정)에 둔다.
> 타깃: 다이어터 · hub 실험 #7

## 한 줄 컨셉
야식 충동이 올 때 버튼을 누르면 그 배달비만큼 가상 저금통에 쌓인다. "참은 돈으로 에어팟 사기".

## 관심 끄는 축 (주력 = 리텐션)
- 🔁 **리텐션(주력): 스트릭 + 목표 게이지** — "참은 돈 87,000원 = 에어팟까지 62%". 참을 때마다 와서 눌러야 쌓이는 구조라 재방문이 곧 사용법.
- 🚀 바이럴(보조): "N일째 야식 참는 중 / 참은 돈 XX원" 현황 카드 공유.

## 유입(모집) 전략
- 릴스 훅: 밤 11시 배달앱 켠 오리 → 갈등 → 버튼 누름 → 저금통에 돈 떨어지는 애니메이션 (결과 먼저)
- 공감 포인트: "이번 달 배달비" (w04)에서 충격받은 사람들의 다음 행동으로 연결 — w04 앱에서 이 앱으로 크로스 링크
- 챌린지 프레이밍: "일주일 야식 참기 챌린지" 스토리 리그램 유도

## 핵심 기능
- "참았다" 버튼: 참은 야식 종류 선택 → 해당 평균 배달비만큼 적립 (치킨 25,000 / 족발 35,000 등)
- 목표 설정: 갖고 싶은 물건 + 가격 → 진행률 게이지
- 스트릭: 연속 참은 날, 최고 기록, 7일/30일 배지
- "무너짐" 버튼도 있음 (기록 리셋 아님, 정직한 통계) — 자학 톤 유지

## 저장 방식
- 로컬로 충분 (localStorage) — 순간 소비형 + 개인 기록
