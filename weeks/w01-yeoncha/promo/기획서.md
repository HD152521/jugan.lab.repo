## 핵심 메시지 (한 문장)

**"8월 17일 하루, 이 연차 한 개가 6일 연휴가 된다 — 연차술사가 대신 계산해준다."**
(연차 1개당 며칠 = '효율' 숫자와 등급 배지로 "나만 계획 없나?" FOMO를 정확히 찌른다)

## 타깃 + 그들이 멈출 이유

- **1차 타깃**: 연차가 애매하게 남은 20~30대 직장인. "이거 언제 쓰지, 어떻게 붙여야 이득이지"를 매년 손으로 계산하다 포기하는 사람.
- **2차 타깃**: 여름휴가·추석 계획 짜는 커플/친구/동료 무리 (친구 맞추기 기능의 공유 타깃).
- **멈추는 이유**:
  1. *타이밍* — 8/15 광복절(토)→8/17 대체공휴일 이슈 + 추석(9/24~26) 검색 수요가 7월부터 몰리는 시즌 진입 직전 (트렌드리포트 "이번 주 시즌 이슈"). 지금 딱 궁금한 주제.
  2. *직접 질문형 훅* — "연차 며칠 남았어요?"류 호명 훅이 숏폼 전반 고성과 (훅 패턴 TOP5 #1).
  3. *숫자 선공개* — 결과값(연휴 일수)을 먼저 던지는 정보성 릴스는 저장 신호에 유리 (핵심 관찰 #1, 훅 패턴 #2).

---

## 훅 (첫 3초에 보이는 화면 + 자막 카피)

- **화면**: 연차술사 결과 카드 하나를 화면 꽉 차게 클로즈업. 카드 상단에 보라색 **`갓성비 SSS급 🔥`** 등급 배지, 그 아래 초대형 숫자 **`6일`**(card-days, 2.6rem), 옆에 **`D-38`** 배지(accent 빨강), 효율 pill **`연차 1개당 6일`**.
- **정지컷 1.5초 유지** (훅 패턴 #4 화면 텍스트 정지컷) → 손가락으로 '6일' 숫자를 톡 가리키는 제스처.
- **자막 카피(상단 큰 글씨)**: **"연차 딱 1개 → 6일 연휴"**
- **음성/보이스오버**: "8월 17일, 이 하루 안 쓰면 손해예요." (손실 프레이밍, 훅 패턴 #5)

## 장면 구성 (씬별 · 화면에 실제 뭐가 보이는지 / 초)

### 씬 1 — 훅 (0:00~0:03, 3초)
- 위 훅 화면: SSS 등급 배지 + `6일` + `D-38` + `연차 1개당 6일` 클로즈업, 정지 → 숫자 손가락 포인팅.
- 자막: **"연차 딱 1개 → 6일 연휴"**

### 씬 2 — 문제 공감 (0:03~0:07, 4초)
- 화면: 스마트폰 달력 앱을 손으로 스크롤하며 8월을 훑는 실사 컷 (또는 연차술사 진입 전 상단 hero "연차 몇 개로 **며칠**까지 쉴 수 있을까?" 화면).
- 자막: **"광복절이 토요일… 그럼 나 며칠 쉬는 거야?"**
- 보이스오버: "매년 손으로 계산하다 포기했죠." (0~3초 문제 제시, 훅 패턴 #3 / 3막 구조 1막)

### 씬 3 — 실사용 데모 (0:07~0:13, 6초) ★핵심 화면녹화
- 화면(화면녹화 그대로): 연차술사 **연차 개수 입력창에 `1` 타이핑** → 상단 **`연휴 찾기`** 모드가 켜진 상태 → 결과가 좌르륵 렌더.
- 이어서 **모드 스위치를 `🗓️ 플래너`로 탭** → 상단 임팩트 스탯 **"올해 최대 N일 쉴 수 있어요"** 숫자가 크게 뜸.
- **`설·추석 빼고`** 토글을 한 번 눌러 결과가 바뀌는 것까지 보여줌 (실사용감·원본성 신호, 핵심 관찰 #2 / 니치 동향 "실사용 스크린 데모").
- 자막: **"개수만 넣으면 끝. 붙일 자리까지 자동."**

### 씬 4 — 결과 강조 + 캘린더 (0:13~0:17, 4초)
- 화면: BEST 배지 달린 카드의 **캘린더 스트립(strip)** — 금색 공휴일 칸 사이에 빨강 점선 '연차(leave)' 칸 하나가 콕 박힌 그림. "이 하루만 쓰면 이만큼"이 시각적으로 보임.
- 이어서 **`📅 ICS`** 또는 **`🗓️ 구글 캘린더`** 버튼 탭 → 캘린더에 등록되는 컷.
- 자막: **"바로 내 캘린더에 등록 (전부 내 폰에서만, 서버 전송 없음)"**

### 씬 5 — 친구 맞추기 + 완주 반전 (0:17~0:20, 3초)
- 화면: **`📋 내 플랜`** 탭의 **친구 맞추기(friend-box)** — "🔗 플랜 공유" 링크 → "둘 다 쉬는 날" 교집합 문구가 파란 강조로 뜸.
- 자막(반전 수치): **"3개 넣으면? 최대 9일. 동료랑 겹치는 날까지 맞춰줌."**
- 보이스오버: "너 연차 몇 개 남았어?"

## 완주 장치 (마지막 반전·수치)

- 마지막 1.5초에 **"1개 = 6일"에서 "3개 = 9일"로 숫자 점프** — 초반 훅보다 큰 수치를 마지막에 배치해 "그럼 내 연차로는?" 궁금증으로 되감기/재시청 유도 (트렌드리포트 "마지막에 수치·반전" 원칙, 핵심 관찰 #1 완주율 가중치).
- + "친구랑 겹치는 날까지" 예상 못 한 기능(친구 맞추기)을 끝에 공개해 반전감 부여.

## 저장 / 공유 / 댓글 유도 포인트

- **저장**: 씬 4 자막에 **"다음 달 연차 계획 세울 때 다시 보려면 저장"** 명시 삽입 (트렌드리포트 제안 #3, 알고리즘이 저장에 가중). "연차 1개당 며칠" 효율 숫자 = 나중에 쓰겠다 싶은 실용 정보 한 조각.
- **공유**: 친구 맞추기 = "이거 완전 너잖아, 동료 태그" 포인트. 자막 **"동료한테 보내서 같이 날짜 맞춰봐."**

## 리스크 (뻔해질 수 있는 지점과 회피안)

1. **"또 그냥 계산기 광고" 느낌** → 회피: 씬 3에서 모드 전환·토글을 실제로 누르는 화면녹화 원본 소재로 승부. 인스타 2026 원본성 가중(핵심 관찰/니치 동향) 활용, 스톡·템플릿 컷 금지.
2. **숫자 과장 논란** → 회피: '6일'·'9일'은 실제 앱이 뱉는 값 범위 안에서만 사용하고, 촬영 시 화면에 실제로 뜬 숫자를 그대로 캡처. 효율 안 나오면 "자유롭게"라고 안내하는 정직 포인트를 씬/캡션에 명시해 신뢰 확보.
3. **초반 3초 이탈** → 회피: 설명 자막으로 시작하지 않고 결과 숫자 정지컷부터. 훅 패턴 #2(숫자 선공개)+#4(정지컷) 결합.
4. **길이 밸런스(국문 20~40초 vs 영문 7~15초 상충, 트렌드리포트)** → 회피: 20초로 절충하되 씬 전환 4회로 지속 자극 유지, 지루하면 씬 2를 2초로 압축해 18초까지 단축 가능.
5. **정보성인데 감정 훅 부재** → 회피: "이 하루 안 쓰면 손해"(손실 프레이밍)+마지막 "너 연차 몇 개 남았어?"(호명)로 감정·대화 트리거를 양 끝에 배치.
